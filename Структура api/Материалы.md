Универсальный метод, который будет использоваться для генерации лент с разными типами материалов. 

В него при помощи GET параметров будут передаваться фильтры по разделам, рубрикам, дате. 

При помощи авторизованных запросов (при помощи заголовка авторизации) можно определять показ скрытых материалов (рекомендуется их выделять для понимания что материал скрыт). 

Для конкретного материала стоит вызывать метод соответствующий материалу чтобы получить дополнительные данные.

# API методы

## Список материалов

`GET https://www.irk.ru/api/1/material/materials/`

GET параметры для пагинации:

	page_size - размер страницы
	по умолчанию: 20
	
	page - страница
	по умолчанию: 1

GET параметр для сортировки:
	ordering - параметр сортировки, для сортировки в обратном порядке нужно написать "-" перед именем поля, для сортировки по нескольким полям нужно их перечислить через запятую
	возможные значения: 'id', 'stamp', 'published_time'

GET параметры для фильтрации:

	slug - алиас
	stamp - дата
	category - id рубрики
	category__slug - алиас рубрики
	sites - id раздела сайта (можно указывать несколько раз get параметр для фильтрации по нескольким разделам)
	sites__slugs - алиас раздела сайта (можно указывать несколько раз get параметр для фильтрации по нескольким разделам)
